# Initializing git in your desktop
- Steps :

        - name a folder as "Webspace"
        - open that folder in vs code.
        - in terminal 
                
                git init
                git add .
                git commit -m "initial commit"
                git remote add origin https://github.com/Sandeep-XD/Webspace.git
                git pull origin 
                git push -u origin master
